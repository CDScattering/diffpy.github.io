convertTOFtoD(dtt1, dtt2, zero, tof)

source code  

Convert TOF to d, as
   TOF = Zero + Dtt1 d + Dtt2 d2
-> Dtt2 d^2 + Dtt1 d + (Zero-TOF) = 0

Argument:
- dtt1  :   float
- dtt2  :   float
- zero  :   float
- tof   :   float
